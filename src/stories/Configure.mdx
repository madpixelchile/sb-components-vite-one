    ## Configuración personalizada de estilos
      Deberemos de respetar los nombres de las propiedades entregadas en el objeto de estilos y evitar eliminar propiedades, ya que serán utilizadas por
      el tema. <br/>
      Podremos añadir o reemplazar valores en su interior.<br/>
      Su estructura será muy parecida al siguiente ejemplo:<br/> 

      ```
        //customTheme.js

        export const customThemeColors = {
            breakpoints: {
                ...breakpoints
            },
            gutters: {
                ...gutters
            },
            colors: {
                ...colors
            },
            fontConfig: {
                ...fontConfig
            }
        }

      ```

      ```
        //App.js
        import React from 'react'
        import ReactDOM from 'react-dom/client'
        import App from './App.tsx';
        import { customTheme } from './styles/customTheme/customTheme';
        ReactDOM.createRoot(document.getElementById('root')!).render(
          <React.StrictMode>
            <MainTheme theme={customThemeColors}>
              <App />
            </MainTheme>
          </React.StrictMode>,
        )

      ```

       const CustomClassText = styled(Text)`
